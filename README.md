# Basic React Native Counter App

A simple and clean counter application built using **React Native** and **Expo**. This project demonstrates the fundamentals of mobile UI development, state management using hooks, and cross-platform compatibility.

## üì± Project Overview

This app serves as an introductory project into the world of mobile development. It features a straightforward interface that allows users to increment, decrement, and reset a numeric value.

### Key Features
- **Cross-Platform**: Designed to run on both Android and iOS.
- **Expo Framework**: Built using Expo for rapid development and testing.
- **State Management**: Utilizes React's `useState` hook for managing the counter value.
- **Modern UI**: Simple, responsive layout using React Native's StyleSheet.
